Ichiraku Ramen Bar 🍜

A responsive restaurant website for a fictional Japanese ramen bar inspired by the iconic spot from Naruto. Built as a frontend project to practise responsive layouts, component design, and real-world deployment.

👉 Live Demo: https://ichirakuramenbarsite.netlify.app/

What's on the site

Home — Hero section with the restaurant tagline and call to action

Why Choose Us — Three feature highlights with icons

Menu — 12 dishes with images, names, and descriptions

Gallery — Restaurant ambience photos hosted on Cloudinary

Visit Us — Location, opening hours, and contact details

Footer — Logo, social handle, and a fun ninja discount section


Tech Stack

HTML · Tailwind CSS · Bootstrap 5 · Cloudinary (image hosting) · Netlify (deployment)

Run Locally
bashgit clone https://github.com/Artiverma19/ichiraku-ramen-bar.git
cd ichiraku-ramen-bar
Open index.html in your browser. No install needed.
